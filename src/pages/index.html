Vehicle Rental API Documentation

This is the API documentation for the Vehicle Rental system, which allows users to manage vehicle rentals, reservations, and vehicles.

Rentals

1. Get All Rentals

Retrieve all active rentals.

Endpoint: 
GET /api/v1/rental/rentals
Response: 
[
  {
    "_id": "rentalID",
    "vehicleID": "vehicleID",
    "userId": "userID",
    "returnedAt": null,
    "station": "stationName"
  }
]

2. Rent a Vehicle

Rent a vehicle by specifying a vehicle ID and a user ID.

Endpoint: 
POST /api/v1/rental/rentals/add
Request Body: 
{
  "vehicleID": "abc123",
  "userId": "user789"
}
Response: 
{
  "message": "Vehicle rented successfully",
  "rental": {
    "vehicleID": "abc123",
    "userId": "user789"
  }
}

3. Return a Rental

Return a rented vehicle by its rental ID and specify the return station.

Endpoint: 
PUT /api/v1/rental/rentals/return/:id
Request Body: 
{
  "station": "Central Station"
}
Response: 
{
  "message": "Vehicle returned successfully",
  "vehicle": {
    "available": true,
    "station": "Central Station"
  }
}

4. Get a Rental by ID

Retrieve a rental by its ID.

Endpoint: 
GET /api/v1/rental/rentals/:id
Response: 
{
  "id": "rental_id",
  "vehicleID": "vehicle_id",
  "userId": "user_id",
  "returnedAt": null
}

5. Update a Rental by ID

Update a rental by its ID.

Endpoint: 
PUT /api/v1/rental/rentals/update/:id
Request Body: 
{
  "returnedAt": "2024-01-02T00:00:00Z"
}
Response: 
{
  "id": "rental_id",
  "vehicleID": "vehicle_id",
  "userId": "user_id",
  "returnedAt": "2024-01-02T00:00:00Z"
}

Reservations

1. Create a Reservation

Create a new reservation for a vehicle.

Endpoint: 
POST /api/v1/rental/reservations/add
Request Body: 
{
  "vehicleID": "bike456",
  "userId": "user789"
}
Response: 
{
  "_id": "reservationID",
  "vehicleID": "bike456",
  "userId": "user789"
}

2. Get All Reservations

Retrieve all reservations.

Endpoint: 
GET /api/v1/rental/reservations
Response: 
[
  {
    "_id": "reservationID",
    "vehicleID": "bike456",
    "userId": "user789"
  }
]

3. Cancel a Reservation

Cancel a reservation by its ID.

Endpoint: 
DELETE /api/v1/rental/reservations/:id
Response: 
{
  "message": "Reservation cancelled successfully",
  "reservation": {
    "_id": "reservationID",
    "userId": "user789",
    "vehicleID": "bike456"
  }
}

4. Get a Reservation by ID

Retrieve a reservation by its ID.

Endpoint: 
GET /api/v1/rental/reservations/:id
Response: 
{
  "id": "reservation_id",
  "vehicleID": "vehicle_id",
  "userId": "user_id",
  "createdAt": "2024-01-01T12:00:00Z"
}

5. Update a Reservation by ID

Endpoint: PUT api/v1/rental/reservations/:id

Description: Update a reservation by its ID.

Request Body:

{
    "vehicleID": "new_vehicle_id",
    "userId": "new_user_id"
    }

Response:

{
    "id": "reservation_id",
    "vehicleID": "new_vehicle_id",
    "userId": "new_user_id",
    "updatedAt": "2024-01-02T00:00:00Z"
}

Errors

Common error responses:

{
  "error": "Message describing the error"
}

Vehicles

1. Get All Vehicles

Retrieve all available vehicles.

Endpoint: 
GET /api/v1/rental/vehicles
Response: 
[
  {
    "_id": "vehicleID1",
    "type": "bike",
    "available": true,
    "station": "Hall-29"
  },
  {
    "_id": "vehicleID2",
    "type": "scooter",
    "available": false,
    "station": "Hall-29"
  }
]

2. Get Vehicle by ID

Retrieve details about a specific vehicle by its ID.

Endpoint: 
GET /api/v1/rental/vehicles/:id
Response: 
{
  "_id": "vehicleID1",
  "type": "bike",
  "available": true,
  "station": "Bozolli"
}

3. Add a New Vehicle

Add a new vehicle to the system.

Endpoint: 
POST /api/v1/rental/vehicles/add
Request Body: 
{
  "type": "bike",
  "station": "Bozolli"
}
Response: 
{
  "_id": "vehicleID3",
  "type": "bike",
  "available": true,
  "station": "Hall-29"
}

4. Get a Vehicle by station

Retrieve details about a specific vehicle by its station.

Endpoint: 
PUT /api/v1/rental/station/:station
Request Body: 
{
  "station": "Bozolli"
}
Response: 
{
  "_id": "vehicleID3",
  "type": "bike",
  "available": true,
  "station": "Hall-29"
}

5. Delete a Vehicle by ID

Remove a vehicle from the system by its ID.

Endpoint: 
DELETE /api/v1/rental/vehicles/:id
Response: 
{
  "message": "Vehicle deleted successfully"
}
