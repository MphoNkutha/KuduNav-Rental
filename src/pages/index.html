API Documentation

Rentals

GET /rentals

Get all rentals

Response: Array of rental objects

POST /rentals/add

Rent a vehicle

Request body:

  • vehicleID: ID of the vehicle to rent
  • userId: ID of the user renting the vehicle

Response: Newly created rental object

GET /rentals/:id

Get a rental by ID

Parameters:

  • id: Rental ID

Response: Rental object

PUT /rentals/:id

Update a rental by ID

Parameters:

  • id: Rental ID

Request body: Updated rental data

Response: Updated rental object

DELETE /rentals/:id

Delete a rental by ID

Parameters:

  • id: Rental ID

Response: 204 No Content

Vehicles

GET /vehicles

Get available vehicles

Response: Array of available vehicle objects

GET /vehicles/:id

Get a vehicle by ID

Parameters:

  • id: Vehicle ID

Response: Vehicle object

POST /vehicles/add

Add a new vehicle

Request body:

  • type: Type of the vehicle
  • station: Station where the vehicle is located

Response: Newly created vehicle object

DELETE /vehicles/:id

Delete a vehicle

Parameters:

  • id: Vehicle ID

Response: Deleted vehicle object
